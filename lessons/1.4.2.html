How many things? 1-3, many.

This lesson continues with the counting of things, again using only the minimum numbers/amounts (1, 2, 3, many) needed to show the basics of the system. Keeping the presentation identical to the preceding one should help confirm the pattern for learners. Image files again simply show these numbers: X number of countables together, either as a natural picture or a photoshopping of X number of identical images.
