How many people? 1-3, many.

This lesson introduces counting (of people), using only the minimum numbers/amounts (1, 2, 3, many) needed to show the basics of the system. Image files simply show these numbers: X number of countables together, either as a natural picture or a photoshopping of X number of identical images.
